Gateomi

QR menu → cart → place order → track status. Uses your gateomi_orderkaro backend (/api/public/…).

API base defaults to http://localhost:3000. Override with ?api=https://your-api on any page, or set window.__ORDERKARO_API__ in a script before modules load.

  • Menu Live categories & items (needs API + DB)
  • Cart Review lines, notes, place order
  • Order placed After checkout (or open with ?orderId=…)
  • Track order Polls order status every 5s (?orderId=…)
